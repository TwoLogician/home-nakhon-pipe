ระบบอินเทอร์เน็ตภายในองค์กร


>> สำหรับองค์กรที่มีระบบอินเทอร์เน็ต
ใช้งานอยู่แล้ว เช่น เป็นหน่วยงานสาขา
ที่ได้เชื่อมต่ออินเทอร์เน็ตกับสำนักงานใหญ่
หรือสถานศึกษาที่ได้อินเทอร์เน็ตฟรี
หากไม่มีการจัดการระบบอินเทอร์เน็ตที่ดี
ก็ไม่สามารถใช้งานอินเทอร์เน็ต
ได้เต็มประสิทธิภาพ
>> บริษัทฯรับวางแผน,ติดตั้งและดูแลระบบ
อินเทอร์เน็ตภายในองค์กรทุกรูปแบบ
>> ออกแบบระบบให้เหมาะสม
กับแต่ละองค์กร
>> มีการติดตั้ง
- ระบบตรวจสอบไวรัส
- ระบบป้องกันผู้บุกรุก
- ระบบจำกัดการใช้งาน
- ระบบใช้งานจากนอกสถานที่
- บริการต่างๆ บนอินเทอร์เน็ต
เช่น เว็บ, อีเมล์ เป็นต้น
   
บริการมาตรฐาน สำหรับการใช้งานอินเทอร์เน็ตอย่างเต็มรูปแบบ ประกอบไปด้วย
>> Standard Internet Service เป็นบริการมาตรฐานของอินเทอร์เน็ต
เช่น DNS, NTP, DHCP, WINS, ฯลฯ
>> Proxy Server เพื่อเพิ่มความเร็วให้กับการใช้งานเว็บ 50 - 200%
>> Network Address Translation (NAT) เพื่อรองรับการใช้งานภายในหน่วยงาน
โดยไม่จำกัดจำนวนเครื่อง
>> Internet Filter สำหรับปิดบริการที่ไม่ต้องการให้บุคลากรในองค์กรใช้งาน
เช่น เว็บโป๊, เกมออนไลน์, โปรแกรมสนทนา, โปรแกรมดาวน์โหลดข้อมูล, ฯลฯ
 
บริการพิเศษ สำหรับองค์กรที่มีความต้องการใช้งานเพิ่มเติม
>> Remote Access Service (RAS) สำหรับให้บุคลากรใช้งานอินเทอร์เน็ต
จากนอกสถานที่ได้
>> Web Server สำหรับเผยแพร่ข้อมูลขององค์กรสู่ภายนอก
>> Email Server ระบบอีเมล์สำหรับบุคลากรภายในหน่วยงาน
ซึ่งใช้งานได้ทั้งแบบ Web-base และ POP3
