System

ภาคนี้ก็เหมือนกับภาค Original ทั่วๆ ไป
ที่จะต้องปราบ Boss 8 ตัว
เมื่อปราบได้แต่ละตัวก็จะได้รับความสามารถของ Boss ตัวนั้นๆ มาใช้
หลังจากจัดการกับ Boss 8 ตัวได้แล้ว ก็จะเจอกับ Big Boss
("ดูด" ความสามารถของ Boss มาใช้)

Ability
ส่วนตัว Megaman นั้น จะมีเฉพาะความสามารถแบบธรรมดา
คือ สไลด์ และชาร์จพลัง เท่านั้น
(สไลด์ กับชาร์จพลัง เท่านั้น)

ไม่สามารถไต่ผนังได้เหมือนภาค X ไม่สามารถชาร์จอาวุธพิเศษได้
และไม่มีเกราะให้เก็บเพิ่มเติมเช่นกัน
แต่จะมี "ร้านค้า" ซึ่งสามารถติดตั้งอุปกรณ์พิเศษเพิ่มเติมได้

Save Game
และที่พิเศษก็คือ ภาคนี้สามารถเซฟเกมได้ โดยไม่ต้องจด Password เหมือนภาคแรกๆ
เมื่อผ่าน Stage ต่างๆ ก็จะสามารถเซฟเกมได้

Player Select
ในส่วนที่แตกต่างไปก็คือ ภาคนี้สามารถเลือกตัวละครได้ 2 ตัว
คือ Megaman และ Bass
ซึ่ง Bass จะมีความสามารถที่แตกต่างจาก Megaman

Item
นอกจากนี้แล้ว ในส่วนหลักก็ยังเหมือนเดิม
คือ เมื่อขีดพลังด้านซ้ายบนของจอภาพ (HP) ลดลงจนหมด ตัวละครก็จะตาย
แต่สามารถเพิ่ม HP ได้ ด้วยเม็ดพลัง ซึ่งจะมีอยู่ตามที่ต่างๆ หรือได้จากการฆ่าศัตรู
และเมื่อได้รับอาวุธพิเศษมาจาก Boss แล้ว ก็ต้องใช้ SP
ซึ่งเมื่อ SP ของอาวุธแต่ละชิ้นลดลง ก็สามารถเพิ่มได้ด้วยเม็ดพลัง SP
และตัวละครจะมีชีวิตจำกัด หากเก็บ 1 UP ได้ ก็จะเพิ่มชีวิตให้อีก 1 ชีวิต
นอกจากเม็ดพลังแล้ว ในภาคนี้จะมีการเก็บ "น๊อต" เพื่อนำไปสร้างอุปกรณ์เพิ่มเติมอีกด้วย
ส่วนเพิ่มเติมของภาคนี้ก็คือ CD หากเก็บได้จะมีข้อมูลของตัวละครในนั้น
