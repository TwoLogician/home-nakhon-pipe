ปุ่ม Hot Key

ปุ่ม Hot Key ช่วยให้การเล่นเกมสะดวกสบายมากยิ่งขึ้น มีปุ่ม Hot Key ให้ใช้ได้ 2 ตำแหน่งด้วยกัน คือ
1. ปุ่ม Hot Key ที่ใช้พวกน้ำยาเพิ่มเลือด เพิ่มมานา ทั้งหลาย หรือ ม้วน เทเลพอร์ต ก็ได้ ทำได้โดยการวาง Item ที่ต้องการลงไปบนช่องว่างด้านล่างขวา ซึ่งมีให้ใช้อยู่ 3 ช่องด้วยกัน วิธีใช้งานก็ทำได้ง่ายๆ โดยการกดปุ่มตัวเลข 1 2 3 ตามลำดับ
2. ปุ่ม Hot Key ใช้ Skill ต่างๆ อยู่ทางด้านล่างซ้าย มีทั้งหมด 4 ช่องด้วยกัน ทำได้โดยการกดปุ่ม T เพื่อเลือก Skill ที่ต้องการ จากนั้นเลื่อนเมาร์ไปบน Skill นั้นๆ แล้วกดปุ่ม F1 หรือ F2 หรือ F3 หรือ F4 ตามต้องการ วิธีใช้ก็คือ เมื่อต้องการใช้ Skill ใดๆ ก็กดเลือกจากปุ่ม F1 ถึง F4 ตามลำดับ แต่หากคิดว่าปุ่ม Hot Key สำหรับใช้ Skill น้อยเกินไป สามารถเลือกใช้ได้อีก 4 ชุด โดยการกดปุ่ม F5 ถึง F8 เพื่อเปลี่ยน Hot Key สำหรับ F1 ถึง F4 ไปใช้ชุดอื่นได้

ด้านซ้ายนี่คือ Skill ลัด ใช้โดยการกด F1 F2 F3 F4 ตามลำดับ หากต้องการเปลี่ยน Skill เป็นอีกชุดหนึ่ง สามารถใช้ F5 F6 F7 F8 เปลี่ยนได้ ด้านขวานี่คือ ปุ่มน้ำยาลัด ใช้โดยการกดปุ่ม 1 2 3 เรียงตามลำดับ

จริงๆ แล้วเกมนี้ จะมีตัวช่วยบอกและสอนการเล่น ซึ่งก็คือ "บาร์ซา" โดย บาร์ซา จะเป็นสร้อยอันแรกที่ติดตัวผู้เล่น จะทำหน้าที่พูดคุย และสอนการเล่นต่างๆ ให้ หากใครรำคาญ ก็สามารถถอดทิ้งได้นะครับ

เรียนรู้การใช้ปุ่มต่างๆ แล้ว ลองไปสำรวจเมืองแรกกันเลย เมือง Cronous
