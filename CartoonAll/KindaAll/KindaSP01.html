SPECIAL FILE 1 เจตนาโหด-15C ภาคปริศนา1
คินดะอิจิ และ มิยูกิ ไปเล่นสกีกัน และพักที่โรงแรมลานสกีสึโนะ
ซึ่งมีสมาชิกคนอื่นๆ พักอยู่ด้วย ดังนี้
โอเนะ ชิซึกะ (27) อ.ที่ปรึกษาชมรมสกี รร.ฟุโด
ซึซึโมริ เอมิ ผู้จัดการชมรมสกี รร.ฟุโด ม.5
อาโค ฮารุโทชิ ชมรมสกี รร.ฟุโด ม.6
ยูคิโอกะ โซเฮย์ ชมรมสกี รร.ฟุโด ม.5
ฮารุตะ ยูโกะ กัปตันหญิงชมรมสกี รร.ฟุโด ม.6
ชิรามิเนะ ทะสึกิ ชมรมสกี รร.ฟุโด ม.6 (ผู้ดูแลอุปกรณ์)

ซึ่งก่อนหน้านี้ ชิบุซาวะ เคย์สุเกะ ซึ่งเป็นตัวเก็งประสบอุบัติเหตุ
และในตอนนี้ก็ยังสลบไม่ได้สติ โดย ชิรามิเนะ ถูกกล่าวว่าเป็นต้นเหตุ
ในคืนนั้น คินดะอิจิ ได้พบกับคุณ ชิรามิเนะ ถูกฝังอยู่ใต้กองหิมะหน้าที่พัก

SPECIAL FILE 1 เจตนาโหด-15C ภาคปริศนา2
กล้องวีดีโอรักษาความปลอดภัย ได้ถ่ายภาพขณะคนร้ายลงมือได้พอดี
คนร้ายสวมชุดยาง และใช้ไม้สกีที่มัดใส่ในถุงตีจากด้านหลัง
จึงสันนิฐานว่า คนร้ายคงต้องมีไม้สกีเป็นของตัวเองอย่างแน่นอน
แต่ถึงอย่างนั้น ก็ยังไม่ทราบตัวคนร้ายอยู่ดี

SPECIAL FILE 1 เจตนาโหด-15C ภาคไขปริศนา
แท้จริงแล้วไม้สกีที่ใช้เป็นอาวุธนั้น ไม่ใช่ไม้สกี แต่เป็นผ้าขนหนูแช่แข็ง
แล้วทำให้มีรูปร่างเหมือนไม้สกี ดังนั้น คนที่ไม่มีไม้สกีและต้องใช้เทคนิคนี้
ก็คือ เอมิ ผู้จัดการชมรมสกีนั่นเอง และเหตุผลอีกข้อหนึ่งก็คือ
เธอไม่เคยเห็นเสื้อของ ชิรามิเนะ ที่สามารถเปลี่ยนสีได้ตามอุณหภูมิ
แต่เธอกลับบอกสีเสื้อได้อย่างถูกต้อง เพราะเธอคือคนร้ายนั่นเอง

SPECIAL FILE 2 ใครฆ่าเทพธิดา 1
ในงานโรงเรียนฟุโด ซากิ ได้รู้จักกับผู้หญิงคนหนึ่งที่เขาชอบ
เธอคือ คามิสึ ซายากะ สมาชิกชมรมศิลป์ รร.ฟุโด ม.5
และมีสมาชิกชมรมคนอื่นๆ อีก คือ
โทโดโรกิ มิวาโกะ สมาชิกชมรมศิลป์ รร.ฟุโด ม.6 (เทพธิดา)
ยามาบุกิ คาโอรุโกะ สมาชิกชมรมศิลป์ รร.ฟุโด ม.6 (เทพธิดา)
ชิบาซาโตะ โจจิ สมาชิกชมรมศิลป์ รร.ฟุโด ม.5
ชิโอมิ ฮะสึเนะ รองประธานชมรมศิลป์ รร.ฟุโด ม.6 (เทพธิดา)
เซนะ โคอิจิ ประธานชมรมศิลป์ รร.ฟุโด ม.6
นาคาสึงาวะ เคนโตะ (25) ที่ปรึกษาชมรมศิลป์ รร.ฟุโด

ในตอนค่ำ อ.นาคาสึงาวะ โดนทำร้ายในห้องชมรม
และผู้ต้องสงสัยก็คือ ซายากะ ที่กลับเป็นคนสุดท้าย

SPECIAL FILE 2 ใครฆ่าเทพธิดา 2
เบาะแสเดียวที่มีก็คือ รูปถ่ายที่ถ่ายภาพรูปปั้นไว้
ซึ่งในห้องปฏิบัติการจะมีรูปปั้นอยู่ทั้งหมด 7 ตัว
และในที่เกิดเหตุรูปปั้นถูกทำลายไปหมดแล้ว
หลังจากที่ คินดะอิจิ สืบสวนทุกคน เขาก็ทราบตัวคนร้าย

SPECIAL FILE 2 ใครฆ่าเทพธิดา 3
สาเหตุที่รูปปั้นยังมีอยู่ครบทั้ง 7 ตัว ทั้งๆ ที่น่าจะถูกใช้ไปแล้ว
เพราะคนร้ายได้ใช้เลื่อยตัดรูปปั้นเทพวีนัสที่เหมือนกันอีกตัวหนึ่ง
แบ่งครึ่งทำให้กลายเป็น 2 ซีก แล้วหันด้านที่ดีให้กล้อง
แต่คนร้ายก็ทำคอนแทคเลนส์หล่นในที่เกิดเหตุ คือคุณ ชิโอมิ

อื่นๆ อีกมากมายของมิยูกิกับฮาจิเมะ
เป็นเรื่องราวตลกๆ ของ คินดะอิจิ

อาเคจิกับคดีฆาตกรรมปริศนา (ภาคปริศนา)
คินดะอิจิ โดนจับฐานเป็นผู้ต้องสงสัยคดีฆ่าคนตาย
ผู้ตายคือ เอะกุม่า ทะดะโอะ (55) ทำงานเกี่ยวกับไฟแนนซ์ (ออกเงินกู้)
ถูกฆ่าตายในรถไฟ ซึ่งในสถานที่เกิดเหตุมีผู้เกี่ยวข้อง ดังนี้
ซึมิโยชิ ชินโกะ (29) หน่วยสืบสวนท้องที่ 1 (ผู้จับกุม คินดะอิจิ)
ชิคะซาวะ นาโอมิ (17) รร.มัธยมปลายฟุโดกะคุอิน ม.5
โนะชิโร่ โอโตะฮิโกะ (45) พนักงานบริษัท
คุนิโต ไดสุเกะ (26) ประกอบอาชีพอิสระ
ยากิ คิโยชิ (28) นายตรวจ รถไฟฟุโดซัง
คาเนะโกะ โยชิคะซึ (46) คนขับรถไฟ รถไฟฟุโดซัง

แต่คำให้การของทุกคนกลับตรงกัน คินดะอิจิ จะทำอย่างไร

อาเคจิกับคดีฆาตกรรมปริศนา (ภาคไขปริศนา)
คนร้ายก็คือ ยากิ คิโยชิ คนตรวจตั๋วนั่นเอง
อาเคจิ ทราบเพราะคำให้การของเขาขัดแย้งกันเอง
คิโยชิ ที่ต้องเดินมาจากตู้ 3 แต่เขากลับทราบว่าคนเมานั้นหลับมาก่อน
เพราะเขาปลอมตัวเป็น เอะกุม่า แล้วมาขึ้นจากตู้ 1 ก่อนนั่นเอง
แต่จริงๆ แล้ว สารวัตรอาเคจิ ทราบมาก่อนแล้วว่าคนร้ายคือใคร
เพราะที่ด้ามร่มซึ่งเป็นอาวุธนั้น มีแต่แต่เฉพาะรอยนิ้วมือของผู้ตาย
ฉะนั้นฆาตกรที่ลงมือจะต้องสวมถุงมืออยู่อย่างแน่นอน
เพราะถุงมือทำให้ไม่มีรอยนิ้วมือติดอยู่ที่ด้ามร่ม
หรือหากคนร้ายไม่ได้ใส่ถุงมือ ก็จะต้องมีรอยนิ้วมือติดอยู่ที่ด้ามร่ม
หากลบรอยนิ้วมือออกไป รอยนิ้วมือเดิมของผู้ตายก็จะหายไปด้วย
เพราะฉะนั้นผู้ที่ใส่ถุงมือก็คือ คนตรวจตั๋ว ซึ่งต้องใส่ตามอาชีพอยู่แล้ว
