คินดะอิจิกับคดีฆาตกรรมปริศนา
การ์ตูนเรื่องนี้ผมเริ่มอ่านตอนอยู่ ม.ต้น (ตอนนี้ก็แก่ตัวแล้ว -_-")
ประมาณปี 2537 (1994)
ฮ่านตอนนั้นเล่มแรกๆ คดีฆาตกรรมหมู่บ้านหกมุม
ผมอ่านเจอศพแห้งในโลง โอ้โห... กลัวมากเลยครับ (: P )
เนื่องด้วยส่วนตัวชอบอ่านนิยายสืบสวนเป็นทุนเดิมอยู่แล้ว
(อ่านเชอร์ล๊อกโฮล์ม อะกาทาคริสตี้ ฯลฯ)
จึงชอบเรื่องนี้มาก
ภาพก็สวยดี อ่านแล้วสบายตา
เนื้อเรื่องก็สนุก (ก็ผมชอบนี่นา มันก็ต้องสนุกสิ ฮา)
อ่านตอน คดีฆาตกรรมเกาะมหาสมบัติ
เหมือนเรื่องของ อะกาทาคริสตี้ ตอน ฆาตกรรมยกเกาะ เลยอะ -_-"
แต่ก็ไม่ได้ลอกมานะ แค่คล้ายๆ กัน ฮิฮิ (แต่ของ อะกาทา น่ากลัวกว่าเยอะ -_-|| )

คินดะอิจิ มีอยู่ด้วยกัน 3 ภาคใหญ่ๆ ครับ
คินดะอิจิ กับคดีฆาตกรรมปริศนา เล่ม 1-27
(Kindaichi Shonen-no Jikenbo Vol.1-27)
คินดะอิจิ กับแฟ้มคดีพิศวง เล่ม 1-6
(KINDAICHI Shonen no ...)
คินดะอิจิ กับคดีฆาตกรรมปริศนา CASE 1-7
(Kindaichi Shonen-no Jikenbo CASE 1-7)
และจะมีเล่มพิเศษของ สารวัตรอาเคจิ อีก 2 เล่ม
อาเคจิ กับคดีฆาตกรรมปริศนา
(Akechi keishi no yuuganaru jikenbo)
